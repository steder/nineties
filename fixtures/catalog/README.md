# fixtures/catalog/

Silent audiobook fixtures for AudioBookShelf. Generated by `just fixtures`
(see `generate.py`).

The `.m4b` files are gitignored — they're synthesized locally from a small
ffmpeg Docker container so the repo stays light and contributors don't need
ffmpeg on PATH.

Each fixture has realistic metadata (title, author, narrator) and a 30s
silent audio stream, enough for AudioBookShelf to index correctly and for
our policy/loan tests to exercise real catalog flows.

Aligned with `tests/features/kid_checks_out_book.feature` — book titles
match so scenarios can run end-to-end against this catalog.
